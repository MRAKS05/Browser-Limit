# Frequently Asked Questions (FAQ)

### What Android versions are supported?
- Android 12+ (API 31 and up).

### Is root required?
- No! Uses Shizuku for rootless package management.

### Can I allow Chrome (or other exceptions)?
- Yes, add exceptions and tune heuristic detection from the app’s settings/exception list.

### Will this brick my phone or break Play Protection?
- No actions are taken without explicit user approval for permissions. All removals are local and reversible with correct permissions.

### How can I make persistent config edits?
- Use parental PIN and settings from setup. See the [Getting Started](usage.md) guide.
