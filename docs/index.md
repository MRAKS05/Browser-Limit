# BrowserLimit Docs

BrowserLimit is a next-gen Android utility that enforces digital minimalism by blocking installation of browsers. It uses advanced heuristics, the Shizuku API, and cryptographic controls to secure your environment. 

- 📱 **Platform:** Android 12+
- 🦖 **Engine:** Kotlin, Jetpack Compose, Shizuku
- 🔒 **Security:** Cryptographic parental lock, audit logs, offline-safe cache
- 🤖 **AI:** Google Gemini classification for browser detection

Explore the guides below to get started, understand its unique features, or dig into advanced setup.
