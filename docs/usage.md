# Getting Started

## Installation

1. Download the latest release from the [GitHub releases page](https://github.com/MRAKS05/Browser-Limit/releases).
2. Grant required permissions when prompted (Overlay, Usage Access, etc.).
3. (Optional) Configure the Shizuku API for enhanced features.

## First Launch

- Launch BrowserLimit.
- Complete the initial setup wizard.
- Set your parental PIN and parental-lock settings.
- Enable or disable Gemini-based AI heuristics (optional).

## Blocking Browsers

- Detected browsers will be removed or blocked automatically.
- Use the Exception List to allow specific apps or update AI heuristics.

For advanced device setup or troubleshooting, see [Advanced](infra.md).
